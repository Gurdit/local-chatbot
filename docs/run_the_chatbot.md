## Running the Chatbot

1. Ensure your virtual environment is activated
2. Navigate to the project root directory
3. Run the chatbot using:
   ```bash
   python chatbot.py
   ```

## Troubleshooting

- If you encounter package-related errors, verify all required packages are installed:
  ```bash
  pip list
  ```
- For memory-related issues, ensure your system meets the minimum requirements
- For model loading issues, verify all model files are present in the correct directory
